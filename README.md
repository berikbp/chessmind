# ChessMind

> Современная шахматная платформа для обучения с стабильной игрой, сохранением партий, разбором Stockfish, онлайн-матчами с другом, пазлами и AI-коучем, который объясняет, почему важный ход был ошибкой.

ChessMind - это не просто шахматная доска. Это продуктовый прототип для начинающих и игроков среднего уровня, которые хотят играть, сохранять партии, разбирать ошибки и понимать человеческую причину оценки движка.

**Обещание продукта:** сыграть партию, аккуратно её пересмотреть, проанализировать с Stockfish, а затем спросить коуча, что нужно запомнить в следующий раз.

## Почему это больше, чем просто шахматная доска

Большинство шахматных демо заканчиваются на перемещении фигур. ChessMind устроен как настоящий продукт:

- **Играть:** AI-партии, pass-and-play, онлайн-игра по invite, контроль времени, стабильный UX доски.
- **Улучшаться:** сохранённый replay, Stockfish-разбор, классификация ходов, объяснения AI-коуча.
- **Возвращаться:** рейтинги, победы, поражения, история партий, профиль, лидерборд по городам.
- **Монетизировать:** Pro-модалка, разблокировка через промокод, дневные бесплатные лимиты для пазлов и коуча.
- **Масштабировать позже:** Stripe-платежи, кастомный SMTP, более большая база пазлов, матчмейкинг, более богатая память коуча.

## Реализованные Функции

### Правила и управление шахматами

- Полная проверка легальности ходов через `chess.js`.
- Рокировка, взятие на проходе, превращение пешки, шах, мат, пат и ничья.
- Drag-and-drop и клик-ходы.
- Диалог превращения пешки для партий и пазлов.
- Стабильный desktop-лейаут доски с видимыми таймерами и карточками игроков.

### AI-противники

- Easy AI использует локальный minimax с alpha-beta pruning.
- Medium и Hard используют Stockfish 18 Lite WASM в browser worker.
- Сложность и контроль времени выбираются до старта партии.

### Сохранение партий и replay

- Партии сохраняются в Supabase с PGN и метаданными результата.
- Последние партии отображаются в профиле.
- Клик по недавней партии открывает сначала чистую replay-доску.
- Stockfish-анализ и AI-коуч доступны из replay/review-потока.

### Разбор Stockfish

- Полный engine review партии.
- Метки ходов: Best, Good, Inaccuracy, Mistake, Blunder, Missed Win.
- Отображение точности.
- График оценки и eval bar.
- Аннотированный список ходов с навигацией по клику.

### AI-коуч

- Объяснения от OpenAI для выбранных review-моментов.
- Объясняет, почему ход был плохим, какую идею пропустили и какой план помнить.
- Использует Stockfish-моменты как источник истины.
- Умеет переживать неполные ответы AI через fallback-объяснения, чтобы панель не ломалась, если модель пропустит момент.

### Онлайн-мультиплеер

- Создание invite-ссылки.
- Второй авторизованный пользователь заходит по ссылке.
- Ходы синхронизируются через Supabase Realtime.
- Polling fallback поддерживает игру, если realtime задерживается.
- Показывается статус realtime-соединения.
- Реализованы сдача партии, завершение игры, сохранение результата, ссылка на review и rematch-flow.

### Пазлы и дневные лимиты

- Локальный MVP-набор пазлов.
- Бесплатные пользователи могут запускать ограниченное число пазлов в день.
- Pro-пользователи безлимитны.
- Стартовое окно пазла объясняет правила до расходования попытки.
- Реализованы состояние решения, следующий пазл, поддержка превращения пешки и окно конца набора.

### Профиль и социальный слой

- Supabase auth и пользовательские профили.
- Рейтинг, победы, поражения, винрейт, город.
- Таблица последних партий.
- Направление на глобальный и городской лидерборд.

### Pro-направление

- Pro-модалка с заглушкой цен.
- Ввод промокода.
- Демо-промокод: `qwerty123`.
- Дневные лимиты уже связаны с продуктовой логикой.

## Переменные Окружения

Создай `.env.local`:

```bash
NEXT_PUBLIC_SUPABASE_URL=your_supabase_project_url
NEXT_PUBLIC_SUPABASE_ANON_KEY=your_supabase_anon_key
SUPABASE_SERVICE_ROLE_KEY=your_supabase_service_role_key
OPENAI_API_KEY=your_openai_key
```

## Настройка Supabase

Приложение ожидает такие таблицы и функции:

- `profiles`
- `games`
- `rating_history`
- `leaderboard_view`
- `daily_usage`
- `online_games`
- RLS-политики для пользовательских данных
- Supabase Realtime, включённый для `online_games`

Запусти SQL-файлы из репозитория в Supabase SQL Editor:

```text
supabase/fix_handle_new_user.sql
supabase/step20_daily_usage.sql
supabase/step22_online_games.sql
```

Если PostgREST сразу не видит новую таблицу, выполни:

```sql
notify pgrst, 'reload schema';
```

## Примечание по Авторизации

Для демо подтверждение email намеренно отключено в Supabase. Встроенный email sender Supabase ограничен по частоте, поэтому письма подтверждения могут блокировать тестирование после нескольких регистраций.

Рекомендация для продакшена:

- Включить подтверждение email.
- Настроить кастомный SMTP через Supabase.
- Сделать письма подтверждения и сброса пароля надёжными.

## Позиционирование Продукта

ChessMind рассчитан на начинающих и игроков среднего уровня, которым нужен post-game разбор без перегруженного интерфейса анализа. Продуктовая логика coach-first: сыграть партию, сохранить её, пересмотреть, проанализировать, а затем понять человеческую причину ошибки.

Это стартап-подобный прототип шахматного обучающего сервиса: здесь есть геймплей, удержание, социальный слой, AI-обратная связь, онлайн-мультиплеер и монетизационные hooks.
